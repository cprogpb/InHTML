html学习笔记

html(hypetext make-up language)是一种创建网页的标准标记语言,他通过一系列的标签和属性来定义网页的结构和内容！

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。

HTML 标签是由尖括号包围的关键词，比如 <html>

HTML 标签通常是成对出现的，比如 <b>和 </b>

标签对中的第一个标签是开始标签，第二个标签是结束标签,开始和结束标签也被称为开放标签和闭合标签

代码中<!DOCTYPE> 声明为html5文档,代表使用html5标准

代码中<html>它是HTML文档的顶级元素,所有其他HTML元素都必须是它的后代。 标签定义了文档的开始和结束,并将整个HTML内容包裹在其中。标签可以包含全局属性,例如lang属性,用于指定文档的语言(如lang="zh"表示中文)。这对于屏幕阅读器和搜索引擎优化(SEO)非常重要

代码中<head>标签用于定义文档的头部信息,它是所有头部元素的容器。头部信息包括文档的标题、字符集、样式、脚本和其他元信息。这些信息不会直接显示在页面上,但对浏览器和搜索引擎非常重要

常见的<head>标签元素,<title> / <meta> / <link> / <style> / <script> / <base>这些元素共同构成了HTML文档的头部信息,帮助浏览器正确解析和显示网页内容,并为搜索引擎提供必要的元数据.

以上都是不显示在用户界面的内容,下面是可见内容

代码中<body>元素包含了可见的页面内容

代码中<div>=divsion 标签是HTML中用于定义文档分区或部分的标签。它是一个块级元素,通常用于将文档分割为独立的部分,以便通过CSS进行样式设置或通过JavaScript进行操作

代码中<h1>元素定义一个大标题

代码中<p>元素定义一个段落文本

代码中<img>元素定义一个图像标签,后面跟随的src 属性是 HTML 中img标签的一个必需属性, 用于指定图像的 URL可以是本地也可以是网络路径,alt属性是图片显示失败后呈现的文字,height属性是定义图片的显示高度,width属性是宽度

代码中<a>元素定义一个链接,后面跟随的href属性用于定义超链接的目标地址

代码中标签的style属性是定义这个标签的显示效果拥有各种属性,例如高/宽/内外边距/行距/字体大小等等
